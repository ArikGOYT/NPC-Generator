# 🎭 NPC Generator

Простой консольный генератор персонажей для тех, кому лень придумывать имена и секреты для своих миров.

### 🚀 Быстрый старт
1. Склонируй этот репо:
`git clone https://github.com/ArikGOYT/NPC-Generator.git`
2. Запусти основной файл:
`python main.py`

### 🛠️ Как это работает
Прога берет данные из `npc_data.json` и рандомит из них комбо. Хочешь добавить новых черт или ролей? Просто отредактируй JSON-файл — никакой перекомпиляции не требуется.

### 📦 Стек
- Python 3.x
- JSON (для хранения базы знаний)
